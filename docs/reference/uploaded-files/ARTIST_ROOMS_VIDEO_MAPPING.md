## Video Generation Strategy

### Placeholder Videos Currently Used
All 13 rooms currently use placeholder video paths. These need to be replaced with actual 5K cinematic backdrops.

### Generation Requirements
- **Resolution**: 5120 x 2880 (5K) or 4096 x 2304 (4K minimum)
- **Duration**: 30-60 seconds (looping)
- **Frame Rate**: 24fps (cinematic) or 30fps
- **Codec**: H.265/HEVC for efficient streaming
- **Format**: MP4 or WebM
- **Hosting**: S3 CDN via `/manus-storage/` paths

### Generation Tools Available
- **Higgs Field API**: Ultra-high-fidelity 5K video generation
- **Claude API**: Detailed prompt engineering for period-accurate environments
- **Eleven Labs**: Ambient audio generation (separate from video)
